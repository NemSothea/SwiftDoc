# Git command(example:)
1. check status files : git status
2. Stage your changes (select files to commit)
`git add Product.java ProductTest.java or git add . `

3. Commit your changes to local history with a message
`git commit -m "Complete Product class lab activity"`

4. Commit your changes to local history with a message
`git status`

5. Push your local commits to GitHub
`git push origin develop`
